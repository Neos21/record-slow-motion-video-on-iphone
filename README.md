# Record Slow Motion Video On iPhone

iPhone でスローモーション動画を撮影するための Swift 4 コード片。


## 検証環境

- 動作確認端末 : iPhone6Plus・iPhoneSE・iPhone7Plus・iPhone11ProMax
- iOS バージョン : v11.3 〜 v14.0
- Xcode : v9.3 (9E145) 以降
- Swift : v4.1 (swiftlang-902.0.48 clang-902.0.37.1) 以降
